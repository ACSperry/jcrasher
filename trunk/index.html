Dependencies

JCrasher code links against (depends on) the following:

  • java-getopt-1.0.10.jar (GNU GetOpt) [LGPL].
    • Source release contains lib/java-getopt-1.0.10.jar (sources and binaries).
    • Binary release contains the binaries.
  • fb-crasher subset [Apache License 2.0].
    • Source release contains the sources.
    • Binary release contains the binaries.
  • JUnit 3.8.1 [Common Public License Version 1.0].
    • Not included.
    • Required to run the JCrasher test cases included in the source release.
    • Required to run any test cases generated by JCrasher.
